# QRCodeGenerator
いちいちサーバー側にリクエストを送っている QR コード生成サービスが気に食わなかったので作成。
代替となるWebアプリは存在すると思います。

## 特徴
- 完全にクライアント上で動作します。
- 通信を行う必要がないため高速です。
